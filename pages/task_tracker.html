About

For my capstone class we had a personal development assignment. There were a handful of different options, and I chose to learn about new technology. I was in the Android development class, which was in Java, so I decided to learn Kotlin scripting language. Since anytime I went to look for Kotlin tutorials I kept seeing Kotlin Multiplatform I decided to learn that too. This app was my final project I turned in. Please note only the Android app works.
