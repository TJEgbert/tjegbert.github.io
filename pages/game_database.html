Game Database

  • GitHub Repository

Status: On pause

Summary

A database where users can create entries manually or using the Twitch API for their personal video game entries. As the database grows the sorting and filtering options expand. There's also the option to make a play list from your game entries.

Technology Used

  • Visual Studio 2022
  • Microsoft Access
  • Language: C#, SQL
  • .Net Framework
  • RestEase
  • NewtonSoft.Json

Conquered Challenges

  • API request and working with the response
  • Keeping user API token up to date
  • Making the program work without an API token
  • Getting access to the Microsoft Access file from outside the bin folder

About

I like most gamer adults have a huge back log of games stretching between many different platforms and some are bound to get forgotten. That was the main idea going into making a program where I could enter my game collection and have it easily searchable and updates as I put more games into it. That was until I asked my brother for his opinion since we are in the same boat. This was when the idea of using an API came from to help fill out some of the information about the game for you. Just from adding in the API this went from a 40-hour week project into almost two.
