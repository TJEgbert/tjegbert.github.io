Puzzle Block

  • Version 1.0 Download
  • Version 1.1 Download

Status: Working On

Summary

A game where the goal is to slide blocks to get them into goals. Similar to the style of sliding block puzzles in the game Goof Troop.

Technology Used

  • Engine: Godot
  • Language: GDScript

Conquered Challenges

  • Precise sliding mechanics
  • Multi block player movement
  • Level design and balance
  • Collision with moving blocks
  • Level changing
  • Saving progress

About

This has been a game I've thought about making for a long time. The idea of the game came from the puzzles from Goofy Troop that my brother and I grew up playing. My thoughts were that puzzles like that world work well for a mobile game. That the goal of this project is to get to be on mobile app stores. That is also the reason why currently I'm not making that code available to be viewed. Currently Puzzle Block has 40 levels in version 1.1 and with improvements to workflow over version 1.0.
