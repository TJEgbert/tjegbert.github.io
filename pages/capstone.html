Summary

A web application professors can use to apply for different grants.

Technology Used

  • Visual Studio 2022
  • MySQL
  • Languages: C#, JavaScript, HTML, SQL
  • ASP.Net Core
  • Razor Pages

Conquered Challenges

  • Learning ASP.Net Core, and Razor pages
  • Generative HTML elements with form submission
  • Sending data from the frontend to the backend
  • Working with Git as a team
  • Filling in HTML select elements from the backend after something in a different select element is selected
  • Working with POSTs on the backend.

About

This was my senior project for my bachelor's degree. My professor was part of the RSPG (Research, Scholarship, and Personal Growth) community and noticed how outdated the process was for applying for a grant and funds distribution. So, he assigned my group of five with developing a web application to modernize the process.
