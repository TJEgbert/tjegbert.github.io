Summary

A Pokémon themed math game aimed at kids between the ages of 3 to 10.

Conquered Challenges

  • Passing data between different screens
  • Handling players input
  • Adding images, fonts and sound effects to a program

About

This was an assignment for CS3280 Object Oriented Windows Applications. I was excited to work on this since it aligns with one of my passions, game development. It is rather a simple game, but it is a game, nonetheless. The goal of the assignment was to make a kids math game that would time them and show how many questions they answered right or wrong. Then it needed a theme to go along with it. I, trying to be the cool uncle, made this game for the nieces and nephews who all love Pokémon. So, I decided to go with a Pokémon themed math game.
