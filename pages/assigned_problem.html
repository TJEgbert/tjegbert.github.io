About

This was originally a quick script I wrote in C++ for my Calculus 1 class. The professor would assign every other odd. Being tired from working all day for some reason, it was hard to add four to get the next problem. So, I wrote a quick script to print out to the console every other odd and gave the total number of problems. When I made this, I was in a Windows Application class so to give myself a little more practice with C# I made it into a program as well with a couple of additional features.
